# Changelog

All notable changes to Pet Pocket will be documented in this file.

## [1.0.0] - Initial Release

### Added
- Core pet stat system (hunger, energy, clean, happy)
- Time-based stat decay system
- Pet actions: Feed, Clean, Play, Sleep
- Multiple pets support with active pet selection
- XP and level progression system
- Coins and shop system
- Inventory system with items
- Skin system (classic, ninja, gold)
- Local notifications for low pet needs
- Stat decay on app start and resume
- Material 3 UI with clean, minimalist design

### Technical
- Flutter + Material 3
- Riverpod for state management
- Drift (SQLite) for local database
- flutter_local_notifications for notifications
